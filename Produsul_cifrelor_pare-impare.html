Produsul cifrelor pare(sau impare) ale unui număr întreg

Alegem numărul 873 și inițializăm două variabile:
  • O variabilă numită produsul o vom inițializa cu 1 pentru a putea înmulți cu ultima cifră.
  • O variabilă numită ultima_cifra o vom inițializa cu 0 pentru a putea stoca în aceasta ultima cifră a numărului
Vom folosi while pentru a opri intrarea în buclă în momentul în care numărul ajunge la 0. La intrarea în bucla while vom realiza operația nr % 10 pentru a stoca ultima cifră, apoi verificăm cu ajutorul lui if dacă condiția este adevărată (vom verifica dacă rezultatul operației este 0, adică verificăm dacă cifra este pară )prin folosirea operatorului % .
Dacă condiția este adevărată vom parcurge instrucțiunea din bloc care înmulțește variabila produsul (prima oară are valoarea 1) cu ultima cifră curentă și apoi stochează rezultatul în variabila produsul pentru a continua înmulțirea cu următoarea cifră.
La ieșirea din if vom tăia ultima cifră a numărului nr prin operația nr / 10 iar rezultatul va fi noul nr care va fi verificat la intrarea în bucla while.
Pentru produsul cifrelor impare vom pune condiția if (ultima_cifra % 2 != 0). 

     $nr=878;
     $produsul=1;
     $ultima_cifra=0;
     
     while ($nr>0){ 
         
       $ultima_cifra = $nr % 10;
       if($ultima_cifra % 2 == 0){ 
           
       $produsul = $produsul * $ultima_cifra;
       }
       $nr = floor($nr / 10);
       }

    echo "Produsul cifrelor pare este: ",$produsul;
