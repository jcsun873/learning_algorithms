Testul de număr prim

Un număr prim este un număr natural, mai mare decât 1, care are exact doi divizori pozitivi: numărul 1 și numărul în sine.

Alegem un număr întreg nr = 11 și inițializăm o variabilă numită numar_Divizori cu 0 pentru a putea numerota câți divizori are numărul ales.
Ne vom folosi de for pentru a parcurge toate cifrele de la 1 până la valoarea numărului nostru.
Ca să putem verifica câți divizori are nr ne vom folosi de if pentru a stabili condiția ca restul împărțirii numărului la divizorul d curent să fie 0.
Dacă rezultatul în urma acestei operații este 0 atunci numar_Divizori va fi incrementat.
La ieșirea din for vom utiliza încă un if pentru a verifica dacă numărul de divizori este egal cu 2.
Dacă condiția este adevărată atunci vom afișa mesajul "Numărul este prim" iar in caz contrar vom afișa "Numărul nu este prim". 

           $nr = 11;
	   $numar_Divizori = 0;
	   
	   for($d = 1; $d <= $nr; $d++){
	       
	       if($nr % $d == 0){
	           $numar_Divizori++;
	       }
	   }
	   if($numar_Divizori == 2){
	       
	      echo "Numărul ",$nr, " este prim: ";
	   }else{

	      echo "Numărul ",$nr, " nu este prim: ";
	   }
