Media aritmetică a cifrelor unui număr întreg

Vom alege un număr nr cu valoarea 53698 și inițializăm 3 variabile:
  • O variabilă ultima_cifra cu 0 pentru a stoca ultima cifră a numărului nr
  • O variabilă nr_cifre pentru a numerota câte cifre are numărul nr
  • O variabilă suma pentru a aduna începând cu 0 toate cifrele numărului nr
Vom folosi while pentru a opri intrarea în buclă la terminarea cifrelor numărului nr.
În blocul de instrucțiuni stocăm ultima cifră și apoi o adunăm la suma existentă în variabila suma.
Cu ajutorul variabilei nr_cifre numerotăm de câte ori se intră în buclă și apoi efectuăm operația n/10 pentru a tăia ultima cifră(rezultatul fiind stocat în nr).
La ieșirea din while vom utiliza variabila numită media_aritmetica pentru a afișa suma adunată în buclă împărțită la numărul de cifre conținute de variabila nr(adică valoarea lui nr_cifre). 

           $nr = 53698;
	   $ultima_cifra = 0;
	   $nr_cifre = 0;
	   $suma = 0;
	   
	   while($nr > 0){
	       
	       $ultima_cifra = $nr % 10;
	       $suma = $suma + $ultima_cifra;
	       $nr_cifre++;
	       $nr = floor($nr / 10);
	   }
	   $media_aritmetica = $suma / $nr_cifre;
	   echo "Media aritmetică este: ",$media_aritmetica;
